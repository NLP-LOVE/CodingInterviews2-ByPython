## 项目介绍

- 此项目是《剑指offer》第二版里算法面试题的Python3实现版本。
- 包括数组、字符串、链表、树、栈、队列和哈希表等数据结构的面试题。
- 查找、排序、动态规划、贪心等算法题。
- 收录的都是各大公司常考的题目。

扎实的基本功是成为优秀程序员的前提条件，而这些体现在编程语言、数据结构和算法上，这也是此项目的初衷。帮助自己的同时也帮助各位同学在各自的道路上能够走得更远。
